# traditional-chinese-medicine
一个中医速记宝典
请开发一个医疗知识库微信小程序，包含两个核心模块：

‌个人信息模块‌

展示用户昵称和手机号（两栏布局）
支持双向绑定编辑功能
数据持久化存储
‌功能中心模块‌

主界面采用网格布局展示功能卡片
每张卡片包含功能图片、标题和描述文字
点击卡片跳转到对应功能详情页
‌当前实现优先级：‌
首先开发“伤寒杂病论”功能：

功能详情页显示完整的病例列表和治疗方案
实现病例信息的全文搜索功能
支持按症状、病名等字段查询特定病例
病例详情页包含完整的诊断和治疗方案
‌技术要求：‌

使用微信小程序原生框架
实现响应式布局和流畅交互
采用云开发数据库进行数据存储
设计简洁美观的医疗类UI界面
确保搜索功能的实时性和准确性
‌数据需求：‌
需要包含《伤寒杂病论》的经典病例数据集，每个病例应包含：病例名称、主要症状、诊断分析、治疗方案、用药建议等字段。

请按照微信小程序最佳实践进行开发，确保代码结构清晰、性能优化良好，并提供详细的使用说明。
